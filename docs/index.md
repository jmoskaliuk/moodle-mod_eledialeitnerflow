## Was macht LeitnerFlow besonders?

- Wiederholung statt einmaligem Lernen
- Fokus auf schwierige Inhalte
- Sichtbarer Lernfortschritt
- Kurze, effektive Sessions

## Einsatzszenarien

- Prüfungsvorbereitung
- Wissenssicherung
- Vokabeltraining
- Microlearning
